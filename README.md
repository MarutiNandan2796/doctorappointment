# Doctor Appoints

A Prescripto-inspired doctor appointment booking app built with the MERN stack.

## Features
- Modern landing page with specialty filters and doctor highlights
- Doctor listing, doctor detail, about, contact, login, register, and dashboard screens
- Express API for doctors, auth, and appointments
- MongoDB-ready backend structure with a clean fallback data layer

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start both apps:
   ```bash
   npm run dev
   ```
3. Frontend runs on Vite's default port and the API runs on `http://localhost:4000`.

## Environment
Copy `.env.example` files into `.env` inside `client` and `server` if you want custom configuration.
